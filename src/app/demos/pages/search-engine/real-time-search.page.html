This example shows how to implement real-time search by using different fields.

Notes:

  • the first group demonstrates autocomplete real-time search with a single select, while the cities list is determined by the selected country.
  • the second group demonstrates autocomplete real-time search with multi-select, using the 'autocomplete-multiple' field.
  • the third group demonstrates autocomplete real-time search with multi-select, while the search results are shown within a checklist component.
