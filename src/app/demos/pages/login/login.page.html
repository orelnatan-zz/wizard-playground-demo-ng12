This example shows how to create a basic login form.

Notes:

  • the validations here are triggered by 'when-submitted' (this is also the default behavior).
  • the 'hideSubmit' property is set to true, means the form don't have the default submit button.
  • the 'frameless' property is set to true, means the form has no frame.
  • the 'wizard-clickable-element' attribute defines the 'Register now' string as clickable.
