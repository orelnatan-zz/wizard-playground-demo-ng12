This example shows how to implement custom validations using the 'VALIDATE' event.

Notes:

  • This group obligates the user to choose only two fields by a custom validation.
  • All fields are not required, but they have their own validations that will still take effect if any value is inserted.
  • The third field has an IP address custom validation attached to it.
  • The fourth field has an Israeli ID custom validation attached to it.
